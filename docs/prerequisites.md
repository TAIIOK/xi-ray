# Подготовка роутера

Перед установкой panel убедитесь, что выполнены все пункты ниже. Без них onboarding не завершится или VPN не заработает на гостях.

## 1. Роутер и прошивка

| Требование | Пояснение |
|------------|-----------|
| **Модель** | Xiaomi BE7000 (XiaoQiang) |
| **SSH** | Root-доступ по SSH (`ssh root@192.168.31.1`) |
| **Интернет на роутере** | Нужен для скачивания Xray и подписок |

Проверка SSH:

```bash
ssh root@192.168.31.1
uname -m   # ожидается aarch64 / arm64
```

## 2. USB-накопитель

Panel, Xray и данные хранятся на **USB**, а не во внутренней flash роутера.

| Параметр | Рекомендация |
|----------|--------------|
| Формат | ext4 или другой, поддерживаемый роутером |
| Объём | от 512 МБ (лучше 1+ ГБ) |
| Монтирование | После вставки появится `/mnt/usb-XXXXX/` |

> **Совет по загрузке:** если роутер «зависает» или не выдаёт интернет при **холодной загрузке с вставленной флешкой**, см. [Устранение неполадок → Ошибка загрузки с USB](troubleshooting.md#ошибка-загрузки-роутера-с-usb-флешкой).

После установки USB должен оставаться подключённым для работы panel и Xray.

## 3. Гостевая Wi‑Fi

1. В веб-интерфейсе роутера (`http://192.168.31.1`) или Mi Home включите **гостевую сеть**.
2. Задайте SSID и пароль.
3. Убедитесь, что гости **изолированы** от основной LAN (обычно включено по умолчанию).

Узнать подсеть гостей (через SSH):

```bash
ip addr show
ip route
```

Часто это `192.168.33.0/24` — значение по умолчанию в panel. Если у вас другая подсеть, укажите её в настройках panel или в `panel.json` (`guest_subnet`).

## 4. VPN-подписка или сервер

Нужен хотя бы один рабочий VLESS endpoint:

- URL **подписки** (`https://…`), или
- одиночная ссылка **`vless://…`**

Panel поддерживает также `vmess://`, `trojan://`, `ss://`, но прозрачный прокси для гостей настроен под **VLESS**.

## 5. Сеть и порты

| Порт / адрес | Назначение |
|--------------|------------|
| `192.168.31.1:7777` | Веб-панель (LAN) |
| `127.0.0.1:10085` | Xray API (локально на роутере) |
| `12346` | REDIRECT TCP (гостевой трафик) |
| `12345` | TProxy UDP |
| `10808` | SOCKS (отладка на роутере) |

Убедитесь, что с компьютера в основной сети открывается `http://192.168.31.1:7777` после установки.

## 6. Чеклист перед установкой

- [ ] SSH root работает
- [ ] USB вставлен и смонтирован (`ls /mnt/usb-*`)
- [ ] Гостевая Wi‑Fi включена, известна подсеть
- [ ] Есть подписка или `vless://` ссылка
- [ ] Роутер имеет доступ в интернет
