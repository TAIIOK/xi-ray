# Использование панели

## Onboarding

После установки откройте `http://192.168.31.1:7777/onboarding`.

| Шаг | Действие |
|-----|----------|
| **1. Пароль** | Смените `admin` / `admin` на свой логин и пароль (мин. 6 символов) |
| **2. Пути и USB** | Выберите USB-раздел, проверьте пути к Xray и скриптам; при необходимости нажмите **Скачать Xray** |
| **3. Сервер VPN** | Вставьте URL подписки или `vless://` ссылку, выберите сервер(ы) |
| **4. Завершение** | Включите «Применить конфиг» и нажмите **Завершить настройку** |

Кнопка **Установить недостающее** создаёт каталоги и копирует embedded-скрипты на USB.

## Dashboard

Главная страница показывает:

- статус процесса Xray (работает / остановлен);
- exit IP через VPN;
- счётчики iptables для гостевой подсети;
- версию panel.

## Подписки

Страница **Подписки**:

- добавление URL подписки;
- ручное обновление;
- автоматическое обновление по расписанию (настраивается в `panel.json`).

Поддерживаемые протоколы в ссылках: VLESS, VMess, Trojan, Shadowsocks.

## Серверы

Страница **Серверы**:

| Режим | Поведение |
|-------|-----------|
| **Один сервер** | Один outbound в `config.json` |
| **Несколько + failover** | Несколько outbounds + Xray Observatory + balancer |

После выбора серверов нажмите **Apply + Restart** (или используйте кнопку Apply на dashboard).

### Failover

1. Выберите 2+ nodes на странице **Серверы**
2. Включите **Несколько + failover**
3. **Apply + Restart**

Observatory проверяет outbounds; balancer переключает на живой сервер.

## Apply и Restart

**Apply** выполняет:

1. Генерацию `config.json` для Xray
2. Генерацию `/data/xray-guest-iptables.sh` (или путь из конфига)
3. `xray -test`
4. Перезапуск через `startup_xray_guest.sh`

**Restart** — только перезапуск Xray без перегенерации конфига.

## Маршрутизация

Страница **Маршрутизация** (если включена в UI): правила direct/proxy/block для гостевого трафика, обход частных IP и хостов VPN.

## Настройки

- пути к бинарникам и конфигам;
- пароль panel;
- параметры Observatory (probe URL, interval);
- автообновление подписок;
- self-update panel.

## Проверка VPN

### С роутера (SOCKS)

```bash
curl -x socks5h://127.0.0.1:10808 https://ifconfig.me
```

### С устройства в гостевой Wi‑Fi

Подключитесь к гостевой сети **без** VPN-клиента на телефоне/ноутбуке. Откройте https://ifconfig.me — IP должен совпадать с exit IP VPN-сервера.

### iptables

```bash
iptables -t nat -L XRAY_GUEST_TCP -v -n | tail -5
```

Счётчики пакетов должны расти при активности гостей.

## Логи

В панели: страница **Логи** (panel, startup, xray access/error).

На роутере:

```bash
tail -f /mnt/usb-*/xiaomi-vless/panel.log
tail -f /mnt/usb-*/xiaomi-vless/xray-startup.log
tail -f /data/xiaomi-vless-boot.log
```
