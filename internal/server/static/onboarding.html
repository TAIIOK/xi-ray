Шаг 1 — Смена пароля

Пароль по умолчанию admin небезопасен. Задайте новый (минимум 6 символов).

Шаг 2 — Пути и USB

У каждого роутера свой путь к USB. Выберите раздел или укажите пути вручную.

Кнопка «Скачать Xray» загружает бинарник (linux arm64) и geo-файлы с GitHub в каталог …/xray/ на USB. Нужен интернет на роутере.

Шаг 3 — Гостевая сеть

Panel перехватывает трафик только с гостевой Wi‑Fi. По умолчанию подсеть Xiaomi — 192.168.33.0/24 (шлюз 192.168.33.1 на br-guest). Если у вас другая — укажите и проверьте.

Шаг 4 — Сервер VPN

—

Поддерживаются https://… подписки (несколько серверов) и одиночные ссылки vless://, vmess://, trojan://, ss://.

VLESS серверы из подписки
