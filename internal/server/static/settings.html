Подписки — автообновление

Периодическая загрузка подписок и выбор серверов без ручного вмешательства.

Загружать подписки по URL с заданным интервалом и обновлять список серверов.
После обновления подписки пересобрать config, iptables и перезапустить Xray.

Watchdog (Сторож)

Следит за VPN: при сбое переключает сервер, обновляет подписки и шлёт уведомления.

Периодически проверять работу VPN и реагировать на outage.
При сбое сначала загрузить подписки и попробовать другие серверы из списка.

Fail-open (гостевая Wi‑Fi)

Если VPN или iptables не работают — гости не остаются без интернета.

Снять VPN-правила и пустить трафик гостевой сети напрямую через NAT роутера.
Когда Xray снова заработает — автоматически вернуть маршрутизацию через VPN.
