Справка по matchers

  • geosite:cn — списки доменов v2fly
  • geoip:ru — IP-диапазоны по стране
  • domain:example.com — домен и поддомены
  • full:www.example.com — точное совпадение
  • regexp:.*\\.ads\\. — regex
  • 192.168.0.0/16 — CIDR

Нужны файлы geoip.dat / geosite.dat рядом с Xray.
